## 📌 Projeto em destaque

### Reestruturação de sistema de transporte executivo

Reestruturei um sistema legado do segmento de transporte executivo, recriando a aplicação em uma stack mais moderna com **Laravel + React + Inertia.js + MySQL**.

Atuei de ponta a ponta no projeto, passando por entendimento da operação, remodelagem da base de dados, reconstrução de fluxos administrativos e implementação da nova aplicação.

### Principais desafios do projeto:
- Entendimento das **regras de negócio** existentes
- Reorganização da estrutura da aplicação
- **Remodelagem de dados** e relacionamentos
- Planejamento e execução de **migração de dados**
- Reestruturação de fluxos administrativos internos
- Evolução do sistema sem perder a lógica operacional já utilizada no negócio

### O que esse projeto representa pra mim:
Mais do que apenas trocar tecnologia, esse projeto exigiu compreender profundamente o funcionamento do sistema anterior para reconstruí-lo de forma mais organizada, escalável e sustentável.

É o tipo de trabalho que reforça algo que valorizo muito no desenvolvimento:  
**não basta programar. É preciso entender o processo, a operação e a lógica do negócio.**
